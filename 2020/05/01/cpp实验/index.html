计算捐款总量

【问题描述】
这里需要设计一个捐款人类Donator及一个相关函数getMaxName()，Donator类中包含捐款人的姓名及其捐款额，补充代码，以输出一批捐款人来到前后的捐款总金额，以及本批次捐款人中捐款最高者的姓名，题目保证捐款人数不少于一人。
【输入形式】
第一行为捐款人数及其当前的捐款总额，第二行开始每行为一个捐款人的姓名和个人捐款金额。
样例输入：
3 28.5
Xiaoyu 12
Mike 81.5
Joey 50

输入本批次捐款人到达前后的捐款总额，及本批次中捐款最高者的姓名。
样例输出：
total:28.5
total:172
Mike

电话薄排序(上题类似)

【问题描述】
编写一个程序，输入N个用户的姓名和电话号码，按照用户姓名的词典顺序排列输出用户的姓名和电话号码。
【输入形式】
用户首先在第一行输入一个正整数，该正整数表示待排序的用户数目，然后在下面多行输入多个用户的信息，每行的输入格式为：姓名 电话。以回车结束每个用户的输入。
3
amethystic 1234567
amethyst 654321
wangwei 7645434
【输出形式】
程序输出排序后的结果。每行的输出结果格式也是： 姓名 电话。姓名和电话字段中间没有空格，要求用户姓名不能超过10个字符，超出10个字符时候只取前10个字符作为姓名。电话号码不能超过10位，超过10位时只按10位处理。输出姓名、电话字段各占12个字符宽，输出格式采用默认对齐方式。另外，用户的数量要求不超过50个。
####amethyst######654321
##amethystic#####1234567
#####wangwei#####7645434
（注：其中#号代表空格）

重点：
1、string库中地.size()来处理号码超过10位
2、.substr()两个参数从第几位开始到第几位截止
3、三个字符串常用函数 strcpy(Str1,Str2);把Str2复制到str1中
strcat(str1,str2);把字符串2连接到1地后面
strcmp(sr1,str2);比较str1和str2是否相同

#include <iostream>
#include <string>   
#include <iomanip>  包含stew（）
#define N 20
using namespace std;
class PhoneBook
{
public:
    void input()
    {
        cin >> name >> number;
    }
    void display()
    {
        if (name.size() > 10) //用户姓名超过10位，取前十位字符
        {
            name = name.substr(0, 10);
        }
        if (number.size() > 10)//用户电话号码超过十位，取前十位字符
        {
            number = number.substr(0, 10);
        }
        //占12个字符宽输出
        cout << setw(12) << name << setw(12) << number << endl;
    }
    string getName()
    {
        return name;
    }
private:
    string name, number;
};
void SortBook(PhoneBook s[], int n) //姓名排序
{
    int i, j;
    PhoneBook t;
    for (i = 0; i < n - 1; i++)
    {
        for (j = i + 1; j < n; j++)
        {
            if (s[i].getName() > s[j].getName())
            {
                t = s[i];
                s[i] = s[j];
                s[j] = t;
            }
        }
    }
}
int  main()
{
    PhoneBook  mybook[N];
    int  n;
    int  i;
    cin >> n;
    for (i = 0; i < n; i++)
    {
        mybook[i].input();
    }
    SortBook(mybook, n);
    for (i = 0; i < n; i++)
    {
        mybook[i].display();
    }
    return  0;
}
