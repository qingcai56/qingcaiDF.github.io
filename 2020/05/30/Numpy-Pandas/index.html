Numpy & Pandas

两个科学应用中极为重要的python模块，主要应用有数据分析、机器学习、深度学习等….

优点：1、运算速度快：numpy和pandas都是用c语言编写的，pandas是numpy的升级版本

​ 2、消耗资源少：都是采用矩阵运算，逼python自带的自带和列表快好多

安装：win、macos：pip3 install numpy

​ linux： sudo apt-get install python-numpy

Numpy

Numpy的属性

numpy模块的导入，以及numpy的三个属性

import numpy as np

array.ndim //维度：矩阵的秩

array.shape //行数和列数

array.size //size的个数

Numpy的创建array

创建array有很多属性和很多形式，在建立数组这一方面和matlab操作有许多相似的地方

关键字

array：数组

dtype：指定数据类型

zeros：创建数据全为0

ones：创建数据全为1

empty：创建数据接近0

arrange:指定范围创建

linspace:创建线段

Numpy基本运算

1、四则运算语法和平常一样

2、多少次方用两个乘号表示 如c=b**2

3、Numpy中具有很多的数学函数工具，比如 c=10*np.sin(a) （三角函数）

4、脚本中对print也可以进行一些逻辑判断 如 print（b<3）输出为bool的矩阵

Numpy的矩阵运算：

找最小值操作和matlb语法类似，matlab中一个max（）是找出每一行的最大值，最后的元素个数为列数，要求总体最大即用max(max())

Numpy的索引

numpy的索引就像我们使用二维数组一样，比如我们要打印第三行第二行的那个数：print(A[2][1])
