指针:用来存放内存单元地址的变量

语法形式： 数据类型 *标识符;
地址相关运算： *（解析运算符）与&（取地址运算符）

int &rf; &出现在声明变量左边时表示引用
有时代表声明标识符，有时为运算符
###指针的赋值
(1)存储类型 数据类型 *指针名 = 初始地址
(2)指针名=地址
四类存储类型：auto,static,register,extern

注意事项
1、如果指针使用对象的地址，则对象在此之前需声明
2、可以用一个指针去赋值另一个指针，即多个指针指向同一变量
3、数组名即为一个指针常量（不能被赋值）
4、const int *p1 = ‘&’a;(不能通过指针修改对象值，但可以指向另外对象)
5、int * const p2 = ‘&’a;(不能指向其他变量但可以改变指针值)

指针运算

指针+n 是向当前位置后方n个相同数据的地址
指针-n 是向当前位置前方n个相同数据的地址
指针++和–同理

一般来说指针算数运算与数组搭配使用
指针可以和0进行比较，看指针是否有效（即不指向任何有效的地址）
int *p; p=0 //将p设置为空指针 /习惯：如果不便于把一个有效地址给一个指针变量赋初值，赢当用0作为它的初值， 从而避免不确定地址的执政出现/

用指针处理数组元素

`int array[5];
array = &array[0]
array[3] = *(array+3) //为等价意思

void f(int p[]);
void f(int p[3]);
void f(int *p); //三个函数声明形参列表都等价，即把指向元素类型的指针传过去
`

对象指针名->成员名
例：ptr->getx() 相当于 (ptr).getx();
