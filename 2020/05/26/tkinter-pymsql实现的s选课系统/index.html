​ 大二下学数据库原理这门课，老师要求我们完成一项大作业，参考老师给的示例，自己着手也做了一个选课系统，从python的tkinter学习到python的pymsql数据库的连接，这几天的实验自我摸索感觉知识面拓展了许多😊

一、需求分析

1.1 项目背景

​ 软件名称：cqupt青菜自制选课系统

​ 用户：学生，老师

1.2 任务目标

​ 1.提供学生选课与退课功能

​ 2.提供学生修完课程成绩查询功能

4.3.8 部分UI设计代码

​ Tkinter 模块(Tk 接口)是 Python 的标准 Tk GUI 工具包的接口 .Tk 和 Tkinter 可以在大多数的 Unix 平台下使用,同样可以应用在 Windows 和 Macintosh 系统里。Tk8.0 的后续版本可以实现本地窗口风格,并良好地运行在绝大多数平台中。 tk在window系统中的窗口和按钮是比mac筹太多了，别人都简洁扁平风的感觉window有上世界win98那味。

​ 还是取出来对二维元组result的处理个人觉得有些问题，这部分工作量其实才是最大的，因为不知道到底去了多少组数据，就定义了一个c=len（result），用for循环动态生成frame框，所以我就事先把这个窗口写的特别长，免得数据溢出到窗口外面去了。
